Get your WordPress webiste mobile and desktop score from the Google Pagespeed API to your WordPress dashboard. 

== Description ==

The Speed ​​Checker plugin helps you get your WordPress website page and post scores for both mobile and desktop. This plugin uses the Google PageSpeed ​​Insights API to get results.
get your complete website performance report from [here](https://speedupmywebsite.co)



== Installation ==
1. Log in to your WordPress Admin page, which is often located at http://yourdomain.com/wp-admin.
2. Click the \"Add New\" button on the Plugins screen after navigating there.
3. Locate the Insights from Google PageSpeed zip file by clicking on the \"Upload\" link at the top of the page.
4. After the installation is complete, upload the file and click \"Activate Plugin.\"
5. Congrats; move on to configuration; the installation is finished.
